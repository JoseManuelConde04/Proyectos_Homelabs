#Proyectos Homelabs
En este repositorio voy a documentar y mostrar mi progreso montando **4 proyectos** que van a ir interconectandose entre ellos mismos.
Los proyectos van a ser:
1. Linux + Documentación personal
    -En el aprenderé a usar linux avanzado, adminisración básica y avanzada, bash scripting y gestión de servicios y logs.
    - Por lo que iré subiendo la documentación a mi repositorio de *'Mis_docus_y_manuales'*, pero el proceso del homelab estará aquí totalmente documentado
2. Infraestructura Interconectada
    - Mi objetivo con este homelab es que 3VMs se interconecten entre sí: 
        a) Servidor Web --> Nginx
        b) Servidor de Base de Datos --> PostgreSQL
        c) Servidor de Monitorización--> Grafana y Prometheus
    - Así aprenderé a crear e instalar diferentes servicios, networking entre distintas máquinas, logs, control de recursos....
    -Todo estará documentado de la misma forma una parte de pasos irá a *'Mis_docus_y_manuales'* y el proyecto integro estará aquí.
3. Monitorización y métricas
    -En este mi objetivo es analizar el rendimiento del anterior homelab, usando Prometheus y Grafana
    -Aprenderé sobre como se monitorizan los distintos componentes de un sistema como hacer dashboards y a identificar distintos cuellos de botella
    - Lo que ocurrirá al final de este homelab es que obtendré dashboard exportados más todos los incidentes documentados para que si en un futuro trabajo ocurren pueda darles el soporte y pueda actuar correctamente
4. Fiabilidad / Optimización / SRE
    - En este pondré en práctica todo lo anterior de 0 simulando fallos, analizando latencias, optimizando recursos y mejorando la resiliencia entre otras
    -Aprenderé SLA, tolerancia a fallos, troubleshooting avanzado, optimización incremental de sistemas.
    -El resultado será como en los anteriores lo dejaré todo documentado con experimentos, métricas antes y después y una lista de mejoras y buenas prácticas realizadas
